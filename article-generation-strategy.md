# 📝 明鉴文章生成策略（英文为主）

## 🎯 核心原则

### 1. 英文优先原则
- **所有文章首先用英文撰写**
- 英文内容完整、深度优先
- 中文内容基于英文准确翻译/适配
- 英文为主，中文为辅

### 2. 内容质量原则
- **英文内容深度优先**：确保技术深度和准确性
- **中文内容准确对应**：确保与英文内容准确对应
- **质量一致性**：中英文内容质量保持一致
- **专业准确性**：确保技术内容的专业性

### 3. 生成流程原则
```
英文内容生成 → 英文质量验证 → 中文内容生成 → 双语对应验证 → 发布
```

## 🔄 生成流程详细设计

### 步骤1：英文主题选择
```bash
# 从英文主题库选择主题
EN_TOPIC = select_from(EN_TOPICS)
```

### 步骤2：英文内容生成
```bash
# 生成完整英文内容
EN_CONTENT = generate_english_content(EN_TOPIC)
```

### 步骤3：英文质量验证
```bash
# 验证英文内容质量
if verify_english_quality(EN_CONTENT):
    proceed_to_next_step()
else:
    regenerate_english_content()
```

### 步骤4：中文内容生成
```bash
# 基于英文内容生成中文内容
ZH_TOPIC = translate_topic(EN_TOPIC)
ZH_CONTENT = generate_chinese_content(EN_CONTENT, ZH_TOPIC)
```

### 步骤5：双语对应验证
```bash
# 验证中英文内容对应
if verify_bilingual_correspondence(EN_CONTENT, ZH_CONTENT):
    save_articles()
else:
    adjust_chinese_content()
```

### 步骤6：文件保存
```bash
# 保存中英文文章
save_english_article(EN_TOPIC, EN_CONTENT)
save_chinese_article(ZH_TOPIC, ZH_CONTENT)
```
